# 소비자 피해유형 분석 기반 자동응답 모델

## 1. 프로젝트 개요
본 프로젝트는 소비자 상담 데이터를 기반으로 피해 유형을 자동 분류하고,
분류 결과의 신뢰도를 계산하여 적절한 자동응답을 제공하는 AI 기반 상담 지원 모델이다.

반복적으로 발생하는 소비자 피해 문의를 자동 처리함으로써
상담 인력의 부담을 줄이고, 일관성 있는 대응 품질을 유지하는 것을 목표로 한다.

---

## 2. 개발 기간 및 인원
- 기간: 2024.09 ~ 2024.11
- 인원: 개인 프로젝트

---

## 3. 주요 기능
- 소비자 상담 텍스트 입력
- 피해 유형 자동 분류 (딥러닝 기반)
- 예측 결과 신뢰도 계산
- 신뢰도 기준에 따른 자동응답 또는 상담원 연결
- 피해 유형 통계 및 데이터 분포 분석

---

## 4. 사용 기술 스택

### AI / 데이터 분석
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy

### 시스템 구성
- 피해유형 분류 모델
- 신뢰도 기반 응답 결정 로직
- 자동응답 출력 모듈

---

## 5. 데이터 구성
- 소비자 피해 상담 텍스트 데이터
- 주요 피해 유형
  - 환불 및 취소 문제
  - 배송 지연 및 미배송
  - 제품 불량
  - 서비스 불만
  - 계약 및 약관 분쟁

데이터는 전처리를 거쳐 학습 / 검증 / 테스트 데이터로 분리하였다.

---

## 6. 시스템 구조
[사용자 입력] -> [텍스트 전처리] -> [피해유형 분류 모델] -> [신뢰도 계산] -> [자동응답 결정] -> [응답 출력 또는 상담원 연결]

---

## 7. 모델 학습 및 성능
- 다중 클래스 분류 모델 사용
- Accuracy 지표 기반 성능 평가

### 모델 특징
- 피해 유형 간 데이터 불균형 고려

---

## 8. 문제 해결 경험

### 문제 1. 피해 유형 데이터 불균형
- 문제: 특정 피해 유형에 데이터가 집중되어 모델 편향 발생
- 해결: 클래스 가중치 적용 및 데이터 정규화
- 결과: 소수 클래스 분류 성능 개선

### 문제 2. 잘못된 자동응답 위험
- 문제: 낮은 신뢰도의 예측 결과에도 자동응답 출력
- 해결: 신뢰도 기준 로직 도입
- 결과: 오응답 감소 및 시스템 신뢰성 향상
